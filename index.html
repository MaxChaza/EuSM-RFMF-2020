Interprétations de signatures métaboliques

Au travers des réseaux métaboliques.

Au cours de mes 3 années chez MedDay Pharma nous avons créé et optimiser le processus d'aide à l'interprétation de signatures métaboliques.
Ce processus utilise plusieurs outils que j'ai développé en étroite collaboration avec l'équipe de bioinformaticiens et de biologistes (incluse dans l'équipe MeX à l'unité ToxAlim) dirigé par Fabien Jourdan.

  • Category: Interprétation

MetExplore

Analyse des données omiques dans le contexte des réseaux métaboliques.

[MetExplore V2] offre une solution en gratuite ligne composée d'outils intéractifs pour l'analyse des réseaux métaboliques, l'exploration du réseau et l'analyse des données en omique.
Il est notamment possible de d'importer, de stocker et d'annoter les réseaux métaboliques dans un environnement collaboratif.
L'exploration des réseaux est également facilitée dans MetExplore par un système de tableaux intéractifs reliés à un puissant module de visualisation des réseaux.
Enfin, la contextualisation des éléments métaboliques dans le réseau et le calcul de statistiques de surreprésentation permettent d'interpréter tout type de données en omique.
MetExplore est un projet durable maintenu depuis 2010 en libre accès à https://metexplore.toulouse.inrae.fr/metexplore2/ .

  • Client: Web
  • Category: analyses omiques
  • Refs: [MetExplore V2], [MetExplore V1]

MetExploreViz

Visualisation de réseau métabolique.

[MetExploreViz] fournit une visualisation interactive des réseaux métaboliques afin d'exploiter les données métabolomiques (et autres "omiques").

  • Client: Web
  • Category: Visualisation
  • Refs: [MetExploreViz] web component for interactive metabolic network visualization, Bioinformatics, Maxime Chazalviel et al. http://doi.org/10.1093/bioinformatics/btx588

MetaboRank

Centralité dans le contexte de l'analyse de signature métabolique.

Au vu de la complexité du réseau, il est nécessaire de prioriser les metabolites à explorer.
De ce fait le Dr. Frainay à implémenté un calcul de centralité, le MetaboRank, adapté aux réseaux métaboliques inspiré du Page rank de Google (permettant d'ordonner les sites web suggéré par son moteur de recherche).
Le MetaboRank est calculé à partir d'une sélection de métabolites.
Ce rang permet de prioriser les métabolites à explorer dans le cadre de l'interprétation d'une signature ou encore dans le cadre de la validation de l'extraction de données expérimentales lors de la création de la signature.

  • Langage: java
  • Refs: [MetaboRank] Clément Frainay et al. MetaboRank: network-based recommendation system to interpret and enrich metabolomics results. (2018). Bioinformatics. DOI

MetExplore Tools

Plus loins dans les réseaux métaboliques.

Durant mon contrat chez MedDay en collaboration avec l'INRAE j'ai développé plusieurs outils permettant d'amélioré l'analyse des données de métabolomique sur l'homme.

Extraction de sous réseau en prenant en compte les transitions carbonnées

Cet outil permet d'extraire un sous réseau du réseau métabolique global de l'homme à partir d'une sélection de métabolites.
Le sous réseau généré est constitué des plus courts chemins entre chaque paires de la liste de métabolites sélectionnés.

  • Client: Web
  • Langage: Java, Javascript, Slurm
  • Refs: Clément Frainay, Fabien Jourdan, Computational methods to identify metabolic sub-networks based on metabolomic profiles, Briefings in Bioinformatics. DOI

Reconstruction itérative guidée

Cet outil intéractif permet de reconstruire un sous réseau à partir d'une sélection de noeuds d'intêrets.
Il permet de parcourir le réseau en affichant les voisins d'un noeud lorsqu'on clique dessus.
Au vu de la complexité du réseau, il est nécessaire de prioriser les voisins à afficher. De ce fait le MetaboRank de chaque noeuds est calculé et les noeuds avec les meilleurs rangs sont proposé en priorité.
L'utilisateur peut donc naviguer dans le réseau de manière guidé et/ou totalement libre.

  • Client: Web
  • Langage: Java, Javascript, Slurm
  • Refs: [MetaboRank] Clément Frainay et al. MetaboRank: network-based recommendation system to interpret and enrich metabolomics results. (2018). Bioinformatics. DOI

Mapping intéractif de données longitudinales

Cet outil a été créé dans le cadre d'un projet d'experimentation biologique.
Aux vues du type de données que nous avions, j'ai créé un outils permettant de visualiser les données pour chaque metabolites au court du temps.
Cette visualisation permet donc de voir les variations de manières animé et intéractive.

  • Client: Web
  • Langage: Javascript, SVG

OPAL GUI

Projet tuteuré.

OPAL GUI est un projet que j'ai développé pendant ma 3ème année de licence Informatique. Mes collègues et moi devions développer une IHM pour utiliser la bibliothèque OPAL.
Pour résumer, OPAL (Open Platform Access Library) est une bibliothèque Java open source qui suit les spécifications Global Platform, et offre plusieurs fonctions (manipulation de cap file , gestion de carte à puce,..).
Notre travail était de développer plusieurs fonctions: Processus d'authentification à la carte, installation d'une applet sur la carte, sélection d'une applet sur la carte, suppression d'une applet sur la carte, l'envoi d'APDU à applet.

  • Technologies: Java "Swing",JavaCard, Mercurial
  • Refs: OPAL: an open-source global platform Java Library which includes the remote application management over HTTP
